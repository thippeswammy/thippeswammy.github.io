# EceProject - Electronics & Computer Engineering Workspace

A comprehensive collection of engineering projects spanning Computer Vision, Embedded Systems, and Robotics.

## Major Components

### 1. [Arduino Uno](./Arduino%20Uno/)
The primary workspace for Arduino-based Computer Vision projects.
- **Computer Vision**: Face Tracking, Fire Tracking (YOLOv8), and Gesture Recognition.
- **Hardware Control**: Real-time serial communication between Python and Arduino for motor and LED control.
- **Documentation**: See the detailed [Arduino Uno README](./Arduino%20Uno/README.md).

### 2. [Stereo Camera](./StereoCamera/)
Projects involving dual-camera setups for depth estimation and stereo vision processing.

### 3. [Embedded Platforms]
- **[ESP8266](./ESP8266/)**: WiFi-enabled micro-controller projects and IoT prototypes.
- **[Raspberry Pi](./Raspberrypi/)**: Linux-based single-board computer applications and high-level processing.

### 4. [Prototype Labs]
- **[Test1](./Test1/)** & **[Test3](./Test3/)**: Experimental folders for prototyping new models and testing script variations before integration.

## Project Structure
The workspace is organized to separate high-level software (Python) from low-level firmware (Arduino/C++), with shared resources centrally managed within the respective project suites.
